# writing

Personal blog, built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

Live at: https://steffenmoll.github.io/writing/

## Local development

```
bundle exec jekyll serve
```

Posts live in `_posts/`, named `YYYY-MM-DD-title.md` with `title` and `date` in the frontmatter.
